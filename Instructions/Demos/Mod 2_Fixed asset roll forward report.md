# エクスポート対応の電子申告向けの固定資産ロールフォワード

### 前提条件:電子申告のセットアップ

**固定資産ロール フォワード** レポートは、電子申告 (ER) フレームワークを使用します。 レポートを実行する前に、**固定資産**モデルと**固定資産ロールフォワード**の構成を Microsoft Dynamics の Lifecycle Services (LCS) からインポートする必要があります。 手順については、Lifecycle Services の電子申告コンフィギュレーションのダウンロードを参照してください。

このデモでは、**固定資産ロールフォワード** レポートを実行し、Dynamics 365 Finance から Microsoft Excel にレポートをエクスポートするプロセスについて説明します。

**[固定資産] \> [照会とレポート] \>[トランザクション レポート] \> [固定資産ロールフォワード]** の順に移動します。

**[固定資産ロールフォワード]** レポートには、期間決算、財務諸表、税金報告用の読みやすい Excel 形式の固定資産データの詳細が表示されます。

表示されるページで、レポートの選択内容を入力します。 **[パラメーター]** セクションで、目的のレポート データの **[開始日]** フィールドと **[終了日]** フィールドを選択します。

次に、**[通貨]** を選択します。 **[会計通貨]** を選択します。

**[形式マッピング]** フィールドで、既に使用するように構成されている **[固定資産ロールフォワード構成]** を選択します。 レポートは、電子申告フレームワークを使用してレポートを生成します。 レポートを実行する前に、**固定資産**モデルと**固定資産ロールフォワード**の構成を Microsoft Dataverse リポジトリからインポートする必要があります。 これは通常、管理者が行い、エンド ユーザーが使用できる状態にします。

レポートをフィルター処理するには、ページの次のセクションにある **[含めるレコード]** を選択します。 **バッチ処理**を使用してレポートを **[繰り返し]** 実行するように設定することもできます。

**[OK]** を選択します。

レポートから、設定したパラメーターに基づいて Excel ファイルが作成されるようになりました。

ダウンロードした Excel ファイルを開き、**固定資産ロールフォワード** レポートを表示します。 Excel を操作して、ビジネス要件に最適なレポートを表示することができます。

Excel レポートには、取得、破棄、減価償却などの期間活動とともに、各固定資産の期首および期末残高が表示されます。

値は資産レベルで詳細に表示され、固定資産グループと法人別に集計されます。

このレポートは、電子申告のデータ抽出と書式設定機能の優れた例です。
